โปรแกรมนี้จัดทำขึ้นโดย [ นายภัทรพล จันทร์แก้ว 672110153 ]
---------------------------------------------------------------------------
Date update 27/02/2568
---------------------------------------------------------------------------
Admin.java
คลาสสำหรับจัดการฟังก์ชันของผู้ดูแลระบบ (Admin) เช่น การสร้างและจัดการการ์ด ตั้งค่าการเข้าถึงห้องและชั้น
---------------------------------------------------------------------------
Customer.java
คลาสที่จัดการฟังก์ชันของลูกค้า (Customer) เช่น การดูข้อมูลสิทธิ์การเข้าถึงของตนเอง
---------------------------------------------------------------------------
Manager.java
คลาสที่จัดการฟังก์ชันของผู้จัดการ (Manager) เช่น การเปลี่ยนห้องหรือชั้น โดยใช้ระบบ TimeBasedEncryption เพื่อควบคุมระยะเวลาการเข้าถึง
---------------------------------------------------------------------------
BaseCardDecorator.java
คลาสพื้นฐานที่ใช้เป็นตัวตกแต่ง (Decorator) สำหรับการ์ดประเภทต่างๆ เพื่อเพิ่มความสามารถโดยไม่ต้องแก้ไขโครงสร้างหลักของการ์ด
---------------------------------------------------------------------------
Keycard.java
คลาสหลักที่ใช้จัดการเกี่ยวกับคีย์การ์ด (Keycard) และกำหนดพฤติกรรมทั่วไปของการ์ด
---------------------------------------------------------------------------
ScanCard.java
คลาสที่ใช้สำหรับสแกนคีย์การ์ดเพื่อตรวจสอบสิทธิ์ในการเข้าถึงพื้นที่ที่กำหนด
---------------------------------------------------------------------------
InformationCard.java
เก็บข้อมูลเกี่ยวกับห้องและชั้นที่ผู้ใช้สามารถเข้าถึงได้
- ค่าคงที่ roomAdmin, floorAdmin สำหรับ Admin
- ค่าคงที่ roomCustomer, floorCustomer สำหรับ Customer
- รหัสผ่านของ Admin และ Manager (passwordAdmin, passwordManager)
---------------------------------------------------------------------------
AdminCard.java
คลาสที่สืบทอดจาก Keycard และมีฟังก์ชันเฉพาะสำหรับการ์ดประเภท Admin
---------------------------------------------------------------------------
CustomerCard.java
คลาสที่สืบทอดจาก Keycard และมีฟังก์ชันเฉพาะสำหรับการ์ดประเภท Customer
---------------------------------------------------------------------------
ManagerCard.java
คลาสที่สืบทอดจาก Keycard และมีฟังก์ชันเฉพาะสำหรับการ์ดประเภท Manager
---------------------------------------------------------------------------
TimeBasedEncryption.java
คลาสที่ใช้จัดการระบบเข้าถึงแบบมีเวลาจำกัด โดยการ์ดจะหมดอายุภายใน 1 นาทีหลังจากเปลี่ยนห้องหรือชั้น
---------------------------------------------------------------------------
Framerole.java
คลาสที่จัดการ GUI และกำหนดบทบาทของผู้ใช้ในระบบ
---------------------------------------------------------------------------
Main.java
คลาสหลักของโปรแกรม ทำหน้าที่เป็นจุดเริ่มต้นของการทำงานทั้งหมด
---------------------------------------------------------------------------
